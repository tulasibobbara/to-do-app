# 📝 Modern To-Do List Application

A responsive and modern To-Do List web application built using HTML, CSS, and JavaScript.

---

## 🚀 Features

- ✅ Add tasks
- 🗑 Delete tasks
- ✔ Mark tasks as completed
- 💾 Save tasks using Local Storage
- 📱 Responsive design
- 🎨 Modern glassmorphism UI

---

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
